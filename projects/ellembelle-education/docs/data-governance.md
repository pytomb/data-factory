### Consent Collection Process

1. **Identify Requirement:** Determine consent needs for each source
2. **Prepare Documents:** Create consent forms in local languages
3. **Collect Consent:** Obtain signatures/agreements
4. **Document:** Store consent records in `docs/consent-log/`
5. **Verify:** Cross-reference with data manifest
